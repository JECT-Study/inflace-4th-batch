# inflace-batch

3팀의 YouTube 데이터 수집 및 적재를 담당하는 Spring Batch 프로젝트입니다.

이 프로젝트는 한국(`KR`) 기준 YouTube 카테고리를 동기화하고, 카테고리별 채널을 탐색한 뒤, 채널/영상 메트릭을 주기적으로 수집해 서비스에서 활용할 수 있는 형태로 저장하는 역할을 합니다.

주요 책임은 아래와 같습니다.

- YouTube 카테고리 마스터 데이터 동기화
- 카테고리별 신규 채널 탐색 및 적재
- 채널 통계 최신화 및 이력 관리
- 최근 업로드 영상, 영상 통계, 태그 데이터 수집 및 적재
- 배치 실행 결과 로깅 및 알림 전송
